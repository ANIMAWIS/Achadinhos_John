Description:
Static website for displaying product offers pulled from a Google Sheets spreadsheet. The page loads data via fetch from a public URL (CSV or gviz JSON) and renders cards. Includes search, sorting, and filtering by store/category/price. A mobile-friendly menu and filter panel were implemented.

Key features:
- Dynamic product loading from Google Sheets (CSV or gviz JSON)
- Parse utilities for CSV and gviz formats
- Normalization and mapping of spreadsheet columns
- UI rendering with filtering, searching, and sorting
- Admin area with password prompt ("Batatadoce") that loads the sheet in an iframe and a preview of index.html
- Optionally, an Apps Script integration was discussed to post new entries automatically (code provided but later removed per user choice)
- Responsive CSS with animations and card effects
- Navigation hamburger toggle for mobile

Development level: Beginner to intermediate front-end. No backend; all logic in vanilla JavaScript. Suitable for learning HTML/CSS/JS and integrating with simple external data sources.

1. Open index.html in a browser.
